# Get started

1. Install [iTerm2](https://iterm2.com/)
2. Clone this repo to ~/dotfiles
3. cd into the `dotfiles` directory
4. Run `./setup.sh`

This will:

1. Clone dotfiles
   - .gitconfig
   - .git_completion
   - .zshrc
   - and others
2. Install Homebrew and some packages I'll likely need
   - git
   - gpg
   - gcc
   - and others
3. Install asdf and some tools I'll likely need
   - ruby
   - nodejs
   - postgres
   - redis
